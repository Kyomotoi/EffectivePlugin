# EffectivePlugin
通过 WebSocket 或 HTTP 对 Minecraft Spigot Server 的事件进行上报以及接受，以实现使用其它语言进行 Minecraft 插件的编写。

极大地方便了不熟悉 Java / Kotlin 而又想进行 Minecraft 插件开发的开发者。
